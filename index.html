Django Twitter

Django Twitter Clone is a full-stack web application that recreates the essence of Twitter, offering a streamlined and customizable microblogging experience. Leveraging the power of Django, this project aimed to provide users with a familiar platform for real-time sharing of thoughts, updates, and conversations. I implemented a secure user authentication process to ensure a personalized and safe experience for each user. Users can compose and delete tweets. Each user has a personalized profile page displaying their tweets, followers, and following. Developed a robust follow system, allowing users to connect with others and curate their timeline based on the content they find interesting. Ensured a responsive design for seamless user experience across devices.
